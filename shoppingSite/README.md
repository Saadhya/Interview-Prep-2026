# diff between abstraction and encapsulation
- The primary difference is that abstraction focuses on what an object does by hiding complex implementation details, while encapsulation focuses on how it does it by bundling data and methods into a single unit and controlling access

# diff grid and flex

- grid is a 2d layout system 
- flex is a 1d layout system
- Grid controls placement in both directions, whereas Flexbox excels at spacing and aligning items based on their content.
